# Security

If you believe you’ve found a security vulnerability in this project, please **do not** open a public GitHub issue.

**Preferred:** use [GitHub private vulnerability reporting](https://github.com/leoakok/poke-agents/security/advisories/new) if enabled on the repository.

Include:

- Description of the issue and impact
- Steps to reproduce (if safe to share)
- Affected versions or commit, if known

We’ll work with you on a coordinated disclosure timeline.
